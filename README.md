A responsive Netflix clone built to replicate the look and feel of the original Netflix platform. This project focuses on UI design, layout structure, responsiveness, and front-end interactions. It is intended purely for learning and practice.

## Project Overview

This Netflix Clone recreates the core visual components of Netflix, including the hero banner, navigation bar, and movie/show rows. The goal of this project is to strengthen front-end development skills by cloning a real-world, production-level interface.

## Features

- Netflix-style homepage layout  
- Fully responsive design (mobile, tablet, desktop)  
- Clean and modern UI  
- Smooth hover effects and transitions  
- Structured and readable code  

## Technologies Used

- HTML5  
- CSS3
